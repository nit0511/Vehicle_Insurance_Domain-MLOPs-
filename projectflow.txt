1. Create project template by executing template.py file
2. Write the code on setup.py and pyporject.toml file to import local pakages
   >> Find more about "setup.py and prproject.toml" at crashcourse.txt
3. Create a vertual env, activate it and install the requirements from requirements.txt
   conda create --name project_name python=3.10 -y
   conda activate project_name
   add required modules to requirements.txt
   Do "pip install -r requirements.txt"
4. Do a "pip list" on terminal to make sure your have local pakages installed.
